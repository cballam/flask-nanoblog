About Blog

This is a very simple blogging app made using Flask and based off this tutorial

It can be run on any device that has Python 3.6, and was developed using Python 3.6.2.

The required Python libraries are Flask, flask_bootstrap, flask_sqlalchemy, and flask_wtf. With these libraries installed and the blog files installed, a server can be run by running website.py
